Reports throw, catch and throws exception types that do not derive from ::k::Throwable (or are not ::k::Throwable itself).

In K, only a class type that derives from ::k::Throwable may be thrown or caught; throwing a non-Throwable type is a compile-time error (0x01C0).

The check is best-effort and fails soft: it only runs when ::k::Throwable is resolvable from the current file (the standard library is present/imported, or it is declared locally). Exception types that cannot be resolved, and throw operands whose type cannot be inferred, are left untouched.
